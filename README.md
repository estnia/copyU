# copyU - UOS V20 剪贴板管理工具

基于 PyQt5 + SQLite3 开发的轻量级剪贴板管理器，适用于 UOS V20 (Debian 10) 系统。

## 功能特性

- **自动保存**: 自动监听系统剪贴板变化并保存历史记录
- **多标签页系统**: 支持创建多个自定义标签页，分类管理剪贴板内容
- **固定功能**: 将重要记录固定到自定义标签页，永久保留不受过期清理影响
- **批量操作**: 支持 Ctrl/Shift 多选，批量固定/移除/删除记录
- **选择性粘贴** (`Ctrl+~`): 弹出历史记录列表，支持 HTML/纯文本两种格式粘贴
- **自由调整大小**: 主窗口支持自由调整大小（最小 300x200，最大 800x600）
- **开机启动**: 支持设置开机自启动
- **自动清理**: 程序启动时及每隔1小时自动清理过期记录（固定记录除外）
- **内存控制**: 超过1MB的记录自动跳过
- **UOS深度风格**: 圆角、浅色调、无边框窗口

## 安装

### 方式一：Deb 包安装（推荐，离线可用）

```bash
# 下载 deb 包后安装
sudo dpkg -i copyu_1.4.0-beta_offline_amd64.deb

# 如有依赖问题，运行
sudo apt-get install -f
```

**说明**：v1.4.0-beta 起采用模块化架构，所有依赖已内嵌打包，无需额外安装，完全离线可用。

### 方式二：源码运行

```bash
# 安装系统依赖
sudo apt update
sudo apt install -y python3-pyqt5 python3-pip xdotool

# 安装 Python 依赖
pip3 install -r requirements.txt

# 运行
python3 main.py
```

## 运行方式

```bash
python3 main.py
```

程序启动后会在系统托盘显示图标，右键可：
- 显示/隐藏剪贴板历史
- 设置开机启动
- 清理过期记录
- 退出程序

## 使用说明

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+~` (Ctrl+反引号) | 显示/隐藏剪贴板历史窗口 |
| `Ctrl+C` | 复制（系统自动保存到默认页） |
| `↑/↓` | 在列表中上下选择 |
| `←/→` | 切换标签页 |
| `Enter` | 粘贴 HTML 格式（保留原格式） |
| `Shift+Enter` | 粘贴纯文本格式 |
| `Ctrl+单击` | 多选/取消选择单个项目 |
| `Shift+单击` | 选择连续范围的项目 |
| `双击` | 直接粘贴原格式 |
| `Esc` | 关闭窗口 |
| `右键` | 管理记录（固定/移动/删除） |

### 标签页功能

- **默认页**: 所有新复制的记录自动保存至此
- **自定义标签页**: 可创建最多3个自定义标签页，支持拖放排序
- **固定记录**: 右键记录可固定到任意标签页，固定记录永久保留
- **批量操作**: Ctrl/Shift+单击多选后，右键可批量固定/移动/删除
- **移动记录**: 可将记录从一个标签页移动到另一个
- **标签页排序**: 自定义标签页内记录可拖动排序

### 窗口操作

- **调整大小**: 拖动窗口右侧、下侧或右下角可自由调整大小
- **最小尺寸**: 300 x 200
- **最大尺寸**: 800 x 600

## 配置文件

配置文件位于 `~/.config/copyu/config.ini`，包含以下可配置项：

```ini
[General]
database_path = ~/.config/copyu/clipboard_store.db  # 数据库文件路径
max_age_days = 3                                    # 自动清理天数
max_record_size_mb = 1                              # 单条记录最大大小
cleanup_interval_hours = 1                          # 清理间隔（小时）
hotkey_show = <ctrl>+grave                          # 显示/隐藏快捷键

[UI]
window_opacity = 0.95                               # 窗口透明度
window_width = 400                                  # 窗口宽度
window_height = 300                                 # 窗口高度
max_display_items = 50                              # 最大显示条目数
font_size = 12                                      # 字体大小
```

## 数据存储

剪贴板历史记录保存在 `~/.config/copyu/clipboard_store.db` (SQLite3 数据库)中，包含：
- HTML 内容（保留原格式）
- 纯文本内容
- 时间戳
- 应用程序名称
- 标签页关联信息
- 固定状态

## 技术说明

- **架构**: 模块化分层架构 (core → infrastructure → services → ui)
- **单实例**: QLocalSocket/Server 实现单实例，新实例启动自动激活已有窗口
- **全局热键**: 使用 `pynput` 或 `system_hotkey`
- **键盘模拟**: 使用 `xdotool`，QProcess 完全异步非阻塞（无 waitForFinished 阻塞）
- **数据库 I/O**: SQLite WAL 模式 + Context Manager 统一连接管理，自动事务回滚
- **线程池**: QThreadPool + QRunnable Worker 类，支持 4 线程并行处理
- **任务队列**: 有界队列（maxsize=1000），队列满时自动丢弃并记录警告
- **速率限制**: 滑动窗口速率限制器保护剪贴板（20/s）、热键（10/s）、窗口切换（5/s）
- **监控指标**: MetricsCollector 收集延迟直方图、百分位统计（p50/p95/p99）
- **日志轮转**: RotatingFileHandler 自动轮转，错误日志 5MB×3，指标日志 2MB×2
- **界面风格**: 遵循 UOS 深度设计规范
