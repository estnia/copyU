# copyU - UOS V20 剪贴板管理工具

基于 PyQt5 + SQLite3 开发的轻量级剪贴板管理器，适用于 UOS V20 (Debian 10) 系统。

## 功能特性

- **自动保存**: 自动监听系统剪贴板变化并保存历史记录
- **多标签页系统**: 支持创建多个自定义标签页，分类管理剪贴板内容
- **固定功能**: 将重要记录固定到自定义标签页，永久保留不受过期清理影响
- **选择性粘贴** (`Ctrl+~`): 弹出历史记录列表，支持 HTML/纯文本两种格式粘贴
- **自由调整大小**: 主窗口支持自由调整大小（最小 300x200，最大 800x600）
- **开机启动**: 支持设置开机自启动
- **自动清理**: 程序启动时及每隔1小时自动清理过期记录（固定记录除外）
- **内存控制**: 超过1MB的记录自动跳过
- **UOS深度风格**: 圆角、浅色调、无边框窗口

## 安装依赖

```bash
# 安装系统依赖
sudo apt update
sudo apt install -y python3-pyqt5 python3-pip xdotool

# 安装 Python 依赖
pip3 install -r requirements.txt
```

## 运行方式

```bash
python3 main.py
```

程序启动后会在系统托盘显示图标，右键可：
- 显示/隐藏剪贴板历史
- 设置开机启动
- 清理过期记录
- 退出程序

## 使用说明

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+~` (Ctrl+反引号) | 显示/隐藏剪贴板历史窗口 |
| `Ctrl+C` | 复制（系统自动保存到默认页） |
| `↑/↓` | 在列表中上下选择 |
| `Enter` | 粘贴 HTML 格式（保留原格式） |
| `Shift+Enter` | 粘贴纯文本格式 |
| `双击` | 直接粘贴原格式 |
| `Esc` | 关闭窗口 |

### 标签页功能

- **默认页**: 所有新复制的记录自动保存至此
- **自定义标签页**: 可创建最多3个自定义标签页，支持拖放排序
- **固定记录**: 右键记录可固定到任意标签页，固定记录永久保留
- **移动记录**: 可将记录从一个标签页移动到另一个
- **标签页排序**: 自定义标签页内记录可拖动排序

### 窗口操作

- **调整大小**: 拖动窗口右侧、下侧或右下角可自由调整大小
- **最小尺寸**: 300 x 200
- **最大尺寸**: 800 x 600

## 配置文件

配置文件位于 `~/.config/copyu/config.ini`，包含以下可配置项：

```ini
[General]
database_path = ~/.config/copyu/clipboard_store.db  # 数据库文件路径
max_age_days = 3                                    # 自动清理天数
max_record_size_mb = 1                              # 单条记录最大大小
cleanup_interval_hours = 1                          # 清理间隔（小时）
hotkey_show = <ctrl>+grave                          # 显示/隐藏快捷键

[UI]
window_opacity = 0.95                               # 窗口透明度
window_width = 400                                  # 窗口宽度
window_height = 300                                 # 窗口高度
max_display_items = 50                              # 最大显示条目数
font_size = 12                                      # 字体大小
```

## 数据存储

剪贴板历史记录保存在 `~/.config/copyu/clipboard_store.db` (SQLite3 数据库)中，包含：
- HTML 内容（保留原格式）
- 纯文本内容
- 时间戳
- 应用程序名称
- 标签页关联信息
- 固定状态

## 技术说明

- **全局热键**: 使用 `pynput` 或 `system_hotkey`
- **键盘模拟**: 使用 `xdotool` (异步非阻塞) 或 `pyautogui`
- **数据库 I/O**: 使用 `QThread` 异步处理，支持 WAL 模式
- **任务队列**: 使用 `deque(maxlen=1000)` 限制内存增长
- **界面风格**: 遵循 UOS 深度设计规范
